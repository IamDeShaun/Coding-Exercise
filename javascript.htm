Technical exercise - Javascript portion

Please create a class in styles.css and use javascript to replace the existing class on the element. The event will be triggered by a button click/press.

Directions

  1. In styles.css, create a class named happy.
  2. In your new class, make the text color white and the element's background color #43285d.
  3. Now open up javascript.js
  4. Write a function that is executed when button1 is clicked/pressed that does the following:
    1. Removes the class "sad" from the main element
    2. Adds the "happy" class to the main element
